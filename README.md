# pbertran_Codi_TFG_012017
En aquest repositori s'inclou el codi dels contractes intel.ligents implementats en el TFG: Disseny d'un Smart Contract de micromecenatge sobre Ethereum de l'Universitat Oberta de Catalunya elaborat per Pere Bertran Sole.

El codi està en llenguatge solidity,  per executar els contractes es pot seguir el procediment detallat a la memòria del TFG a la secció 6 Implementació dels contractes.
